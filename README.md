5 portfolio tasks from the programming: concepts to construction 2 module, CIS1112, from my first year at edge hill university.
## How to use
Download the task folder you want onto your computer and open the task folder's helper file, CW1_(1-5), in any java IDE to run it.

Note that all code files were coded using java in Eclipse IDE 4.26.0 for 2022-12.
